# 🎬 Sakila SQL Analysis Project

## 📌 Overview
This project performs SQL analysis on the Sakila Movie Rental Database to extract insights on revenue, customers, and rentals.

## 📂 Project Structure
- schema.sql → Database schema
- data.sql → Sample data (Sakila)
- analysis.sql → SQL analysis queries

## 🚀 Key Analysis
- Revenue analysis
- Customer segmentation
- Rental trends
- Top movies

## 🛠 Tools
- MySQL
- SQL
